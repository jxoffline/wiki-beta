Source code Kéo Xe NPC 1.1.4 (jx linux 8.0 & 6.0?)

Tiêu chí của mình trước sau như 1, là hoàn toàn FREE không giấu diếm hay tiền bạc gì cả. Để cộng động JXDev tiếp tục phát triển. Nên AE không cần pm xin mua gì tùm lum hết. Khi script hoàn thiện thì mình share ngay.

Đây là script đầu tiên mình tự làm. Tặng tất cả AE hội quán.

A. Hướng dẫn cài đặt

1) Download file zip về, giải nén và chép toàn bộ thư mục script vào thư mục script của server

2) Mở file script/global/autoexec.lua trên server và tìm đến dòng:

    function main()

thay đổi thành 3 dòng sau: 

Include("\\script\\global\\vinh\\main.lua")
    function main()
        add_npc_vinh()

hoặc xem file autoexec.lua trong cùng thư mục drive này

3) Xong. Khởi động server và tìm đến gần hiệu thuốc Tương Dương: * gặp Triệu Mẫn để sử dụng simcity * gặp Vô Kỵ để điều khiển kéo xe (+ nhận lệnh bài kéo xe*)

Chú ý: lệnh bài Kéo Xe NPC nhận tại Vô Kỵ có thể không chạy cho bản JX 6. Cần phải tạo lệnh bài theo cách hướng dẫn ở mục phần thông tin thêm bên dưới

B. SimCity

1) Điều chỉnh tỷ lệ đánh nhau hoặc nói chuyện hoặc lọc bỏ các nhân vật theo ID bằng cách sửa file

        \script\global\vinh\simcity\config.lua

2) Dùng lệnh bài của Kéo Xe NPC (hoặc các lệnh bài tương tự) có thể tự chuyển sang trạng thái chiến đấu trong thành và tham gia đánh lộn

C. Kéo xe NPC

1) Nếu muốn tạo lệnh bài gọi xe nhanh đem theo bên mình (thay vì NPC Vô Kỵ) thì vào https://jxoffline.github.io/jxtools/shopbuilder.d/ để tạo thêm 1 vật phẩm với đường link tới file script

    \script\global\vinh\keoxe\main.lua

2) Nếu muốn thay đổi danh sách theo sau (hoặc tạo riêng cho mình) thì sửa file

    \script\global\vinh\keoxe\danhsach_xe.lua

3) Nếu muốn thêm NPC theo sau bằng code thì gọi như sau:

            Include("\\script\\global\\vinh\\main.lua")
            KeoXeNpcManager:start()    
            KeoXeNpcManager:TaoNpc(ID hình , Tên, thongtin)

Ví dụ:

            Include("\\script\\global\\vinh\\main.lua")
            KeoXeNpcManager:start()    
            KeoXeNpcManager:TaoNpc(1557, "LieuThanhThanh", {            
                is_boss = 1,    -- 1 la chu vang, 0 la chu trang
                no_revive = 1,  -- 1 la se ko hoi sinh, 0 la hoi sinh sau khi chet
                canFight = 1,   -- 1 la chien dau, 0 la ko chien dau
                series = 5      -- ngu hanh 0-4 , neu 5 thi ngu hanh tu chon
            })

Cám ơn bác Thịnh Magic Tips đã ra video NPC vận tiêu theo sau từ đó mình phát triển thêm làm nền cho script này.
