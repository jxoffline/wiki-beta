🏆 Nội dung

1. Bản Tiếng Việt

Việt hoá xong tool để xem SPR. Share cho mọi người sử dụng thử. Tool được build từ c++, codeblocks. Vẫn giữ nguyên UI ban đầu.

Pass giải nén là 1

Nếu có báo lỗi thiếu file .dll thì các bạn tải về chép vào cùng cấp với file chạy nhé.

Còn có lỗi nào khác thì comment ở dưới để tui biết chỉnh lại nhé.

P/s Khi sử dụng sẽ có 1 số file .spr hoặc .mpc hoặc .asf không load được (có thể do cách convert các file dạng này quá cũ hoặc mới nên tool ko đọc đc). Các file spr mới nếu dùng tool spr eng được share thì đọc ok.

2. Bản tiếng Anh

Tool này là để xem, chỉnh sửa các file resource của một vài game nhà Kingsoft như là Kiếm Hiệp Kỳ Duyên, Truyền Thuyết Ánh Trăng,… các loại file mà nó xử lý là MPC, ASF, RPC, và tất nhiên là có SPR quen thuộc của chúng ta.

Nhìn hình mình họa thì anh em cũng thấy là nó có thể xem, và đọc thông số của các file hình ảnh kể trên. Tiếc là tính năng chỉnh sửa, thêm hoặc bớt frame, điều chỉnh tốc độ đối với file SPR vẫn chỉ nằm ở dạng preview, chứ chưa sửa và lưu trực tiếp vào file SPR được. Hy vọng có tay to nào sớm phát triển thêm tính năng này cho ae dùng cho tiện, nhỉ.

P/S: anh em nào bị lỗi thiếu DLL thì tải cái này về quăng vào nha libwinpthread-1.dll.
